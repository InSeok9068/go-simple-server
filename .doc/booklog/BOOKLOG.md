# BOOKLOG

### 내가 읽었던 도서 목록
- 도서 정보
  - 오픈 API : https://www.data4library.kr/apiUtilization
  - 도서 정보 (제목, 저자, 출판사, 출판일, 페이지, isbn, 내용)
- 도서 이미지 첨부
- 도서 썸네일 이미지
- 책의 삽입 할때마다 책의 정보를 가지고 자동화된 태그 생성
- 책을 새롭게 삽입할때 마다 나의 소중한 기억 도서관으로 넣는것과 같은 애니메이션
- 리스트된 책을 클릭하면 책의 정보를 보여주는 페이지로 이동
  - 좀 더 감성적인 애니메이션이 같이 추가되면 좋음

### 기억하고 싶은 책 내용
- 간편하게 입력하는 내용
  - 책 페이지 번호
  - 책 내용
- 아침, 저녁으로 책 제목과 함께 기억하고 싶은 내용을 푸시로 보내주면서 기억을 상기시키기

### 독후감
- 책을 읽고나서 독후감을 간단히 작성 할 수 있는 기능
- 책에 대한 별점 평가

### 부가기능
- 내가 매월 얼마나 많은 책을 읽었는지 확인하는 페이지
