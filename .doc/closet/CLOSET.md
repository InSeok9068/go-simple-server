# 옷장 서비스

## 동작성 플로우

1. 옷장 이미지 업로드
   - 타입 : 상의/하의/신발/악세사리
   - 태그 : #캐주얼, #스트릿 ..

2. 이미지 저장
   - SQLite에 BLOB 타입으로 저장

3. 이미지 임베딩
   - AI에게 이미지 임베딩 요청
   - 해당 이미지 임베딩 값 SQLite에 업데이트

4. 이미지 검색
   - 타입, 태그를 활용하여 SQLite에서 우선적으로 데이터 검색

5. 브루트포스 Top-K 검색
   - 검색 된 이미지들을 메모리에 올려서 브루트포스 Top-K 검색
   - 상의/하의/신발/악세사리 그룹별로 Top-K 검색

6. AI 스타일 추천
   - AI에게 Top 임베딩 데이터 전달 후 의상 추천 요청
   - AI가 추천한 데이터를 화면에 표시

7. [선택] 옷 입혀보기
   - 나의 전신 이미지를 같이보내서 AI에게 옷을 입혀보는 요청
   - AI가 입혀본 결과를 화면에 표시
